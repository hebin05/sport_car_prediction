# 🏎 Sports Car Price Prediction using Machine Learning

## Overview

This project is a Machine Learning-based web application developed using **Python** and **Flask** to predict the price of sports cars. The model estimates the price based on important vehicle features such as **Horsepower, Torque, Car Make, and Car Model**.

The application provides an interactive and user-friendly interface where users can enter car specifications and instantly receive the predicted price in both **US Dollars (USD)** and **Indian Rupees (INR)**.

## Features

* 🚗 Sports Car Price Prediction
* 🤖 Machine Learning model using Lasso Regression
* 🌐 Flask-based web application
* 🎨 Attractive and responsive user interface
* 🏎 Dynamic car make and model selection
* 💵 Price prediction in USD
* 🇮🇳 Automatic conversion to INR
* 🖼 PNG image support for enhanced UI
* 📊 One-Hot Encoding for categorical features
* 🔥 Easy to use and suitable for academic projects
